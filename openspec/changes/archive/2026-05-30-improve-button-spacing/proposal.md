# Proposal: Mejorar el espaciado entre botones

## What
Aplicar un margen global (Enfoque 3) a todos los botones de la aplicación (`.report-action-btn`, `.report-action-btn-green`, `.report-ghost-btn`) para evitar que queden pegados cuando se renderizan uno al lado del otro.

## Why
Actualmente, múltiples botones renderizados juntos no tienen separación nativa, lo que reduce la accesibilidad visual (están muy juntos) pero se requiere mantener su tamaño original intacto para no afectar la consistencia del diseño.
